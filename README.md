## Setup environment

```bash
conda create --name bike-ds python=3.9
conda activate bike-ds
pip install numpy pandas scipy matplotlib seaborn jupyter streamlit babel
```

## Run steamlit app

```bash
streamlit run dashboard/dashboard.py
```
